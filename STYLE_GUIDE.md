# Style Guide — Smart Calculator Tools

آخر تحديث: 2026-03-15

## 1) الهوية البصرية
### لوحة الألوان (Tokens)
- `--bg`: خلفية رئيسية دافئة
- `--surface`: سطح أساسي للبطاقات
- `--surface-2`: سطح ثانوي
- `--ink`: نص أساسي
- `--muted`: نص ثانوي
- `--stroke`: حدود خفيفة
- `--accent`: لون إيجابي/إجراء رئيسي
- `--accent-2`: إبراز ثانوي
- `--accent-3`: إبراز ثالث
- `--negative`: قيم سالبة/خسائر

### استخدام اللون
- اللون الأساسي للأزرار والأرقام الإيجابية: `--accent`.
- اللون السالب والنتائج السلبية: `--negative`.
- حدود البطاقات والحقول: `--stroke`.

## 2) الخطوط والطباعة
- العنوان الرئيسي: `font-display` (DM Serif Display).
- نصوص الواجهة: `font-sans` (Space Grotesk).
- أحجام مقترحة:
  - H1: `text-4xl` إلى `text-5xl`
  - H2: `text-2xl` إلى `text-3xl`
  - Body: `text-sm` إلى `text-base`
  - مساعد: `text-xs` مع `tracking` واضح

## 3) التخطيط والمسافات
- الحاوية القياسية: `max-w-5xl` إلى `max-w-6xl`.
- هوامش عمودية متدرجة عبر `.section-pad` ونسختها المدمجة.
- نستخدم `gap-4` / `gap-6` كافتراضي للمجموعات.

## 4) المكونات القياسية
### Input Group
- خلفية: `bg-white/70` مع `border-stroke`.
- شكل: `rounded-2xl`.
- نص توضيحي قبل الحقل.

### Results Card
- خلفية: `bg-white/70`.
- حدود خفيفة + عنوان صغير uppercase.
- قيمة رئيسية بحجم `text-2xl` ووزن `font-semibold`.

### Panels
- `surface` للمدخلات
- `surface-2` للمخرجات أو التباين
- ظلال ناعمة: `shadow-soft`

### Charts
- خط الصفر واضح
- الموجب: `accent`
- السالب: `negative` مع نقش خفيف
- Legend مختصر أعلى الرسم

## 5) حالات التفاعل
- Hover: حركة بسيطة `hover:-translate-y-0.5`
- حالة نشطة للأزرار: خلفية داكنة ونص أبيض
- حقول الإدخال لا تستخدم ألوان صارخة لتفادي التشتيت

## 6) المحتوى داخل الحاسبة
- ملخص مختصر (2–3 جمل).
- أقسام ثابتة: Inputs / Outputs / Assumptions / Tips.
- أمثلة واضحة + جدول + رسم واحد على الأقل.

## 7) الوصولية
- تباين الألوان على الأقل AA.
- قابلية استخدام لوحة المفاتيح في الحاسبات الأساسية.
